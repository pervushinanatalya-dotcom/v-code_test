# Theatre Notify Bot

Telegram-бот для сохранения и управления информацией о спектаклях, которые вы планируете посетить.

## Описание

Бот позволяет:
- Сохранять спектакли с информацией о городе, театре, названии и дате
- Просматривать список всех сохраненных спектаклей
- Экспортировать спектакли в Markdown файлы
- Экспортировать отдельные спектакли через inline кнопки

## Установка и запуск

### Локальный запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.env` в корне проекта с токеном бота
3. Запустите бота:
   ```bash
   python -m app.main
   ```

### Запуск через Docker

```bash
docker build -t theatre-bot .
docker run -d --name theatre-bot --env-file .env theatre-bot
```

## Команды бота

- `/start` - Начать работу с ботом
- `/add_show` - Добавить новый спектакль
- `/my_shows` - Показать список всех сохраненных спектаклей
- `/export` - Экспортировать все спектакли в файл
- `/theatres` - Показать список театров в базе

## Технологии

- Python 3.12
- python-telegram-bot 21.7
- SQLite
- Docker
